System Flow: Users authenticate via role-based login → React frontend sends API requests → Spring Boot Controllers validate and route requests → Services execute business logic (product creation, event logging, ownership transfer) → Repositories interact with PostgreSQL → Secure responses returned to frontend → QR codes generated for instant product traceability

  • Role-Based Access Control: Implemented Spring Security with FARMER, DISTRIBUTOR, RESTAURANT, and ADMIN roles ensuring granular permissions and 403 Forbidden responses for unauthorized actions.
  • Product Lifecycle Management: Complete CRUD operations for food products with event logging (SHIPPED, RECEIVED, QUALITY_CHECK, DAMAGED, SOLD) and secure ownership transfers between authorized entities.
  • QR Code Integration: Each product ID encoded into QR codes enabling instant traceability lookups via mobile scan, providing complete lifecycle history from farm to fork.
  • Three-Layer Architecture: Clean separation with Controllers handling API requests, Services managing business logic, and Repositories handling PostgreSQL interactions via Spring Data JDBC.
